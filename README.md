# aiChildren

aiChildren 是一个 **儿童成长数据平台**。

通过 **成长记录 + 每日任务 + BusyBook实践 + AI成长分析**，  
帮助家长理解孩子的成长轨迹，并持续优化成长路径。

本项目当前为 **产品原型（Prototype）**，实现：

- 微信小程序前端
- Flask 后端 API
- 基础成长记录系统
- 每日任务与打卡
- BusyBook作品上传
- 成长数据看板

---

# 产品核心理念

aiChildren 不只是一个记录工具，而是一个 **成长平台**。

产品围绕三个核心系统构建：

### 成长数据系统

记录孩子成长轨迹。

### 成长任务系统

引导孩子形成持续行为习惯。

### 成长实践系统

通过 BusyBook 等实践活动强化学习。

最终形成完整成长闭环：

任务 → 行为 → 成长记录 → AI分析 → 成长反馈 → 新任务

# 核心模块说明

## 首页（成长看板）

首页用于展示孩子当前成长状态。

主要内容：

- AI成长总结
- 今日任务
- 最近成长记录
- 快捷操作入口

---

## 成长中心

成长数据系统。

用于记录与分析孩子成长轨迹。

主要功能：

- 成长记录
- 成长时间轴
- 成长报告
- AI成长趋势分析
- 课程学习记录

---

## 任务星球

成长行为系统。

通过每日任务与挑战机制，引导孩子形成持续成长习惯。

功能包括：

- 今日任务
- 每周任务
- 打卡系统
- 连续打卡统计
- 成就体系

灵感来源：

Duolingo 的习惯养成机制。

---

## BusyBook工坊

成长实践系统。

通过 BusyBook 等实践内容强化学习。

功能包括：

- BusyBook模板库
- BusyBook创作工具
- 打印模板
- 上传作品
- 作品展示与点赞

未来可以扩展为 **内容创作平台**。

---

## 我的

账户管理系统。

功能包括：

- 孩子管理
- 切换孩子
- 我的课程
- 我的任务
- 我的BusyBook
- 系统设置

# 本地启动

## 启动后端

```bash
cd backend

python3 -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python app.py

服务地址：

http://127.0.0.1:5000


⸻

启动小程序

1 打开微信开发者工具
2 导入 miniapp 目录
3 使用测试 AppID 或无 AppID
4 接口地址配置为：

http://127.0.0.1:5000

注意：

真机运行需要 HTTPS 与合法域名。
